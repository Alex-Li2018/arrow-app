# VisualNode类

## 绘制内容

- 节点圆 node
- 节点名称 label

## 属性

- node节点
- selected 是否被选中
- editing 是否正在编辑中
- internalRadius 内径
- radius 半径： 内径 + border
- outsideComponentRadius
- fitRadius
- background：背景色
- internalVerticalOffset
- internalScaleFactor
- insideComponents
- outsideComponents

## 方法

- draw() 绘制
- boundingBox() 边框
